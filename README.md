# Codex Skills

Personal Codex skills repository.

## Install on macOS or Linux

Run from the repository root:

```bash
./install.sh
```

This copies every skill folder into:

```text
~/.codex/skills
```

## Install on Windows

Run from the repository root in PowerShell:

```powershell
.\install.ps1
```

This copies every skill folder into:

```text
$HOME\.codex\skills
```

## Notes

- The repository stores user skills only.
- Codex system skills from `.system` are not included.
- Restart Codex after installing or updating skills.
